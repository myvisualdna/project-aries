## Amazon Clone (E-commerce project)

Live Demo: https://aries-project-firebase.web.app/

![Screenshot](screenshot.png)

Technology used:

- React
- Firebase (auth & database)
- Redux
- Redux Toolkit
- Redux-Thunk
- Redux Dev Tools
- React-Router
- React-Router-Dom
- SASS
- Material UI
- Material Icons
- React Currency Format
- Stripe (for payments integration)
